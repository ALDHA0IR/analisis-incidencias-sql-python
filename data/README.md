
# Data

Esta carpeta contiene los datos utilizados en el proyecto.

## Subcarpetas
- `raw/`  
  Dataset original sin modificaciones.

- `processed/`  
  Dataset procesado y filtrado con Python, listo para análisis y visualización.

## Notas
- Los datos procesados se generan a partir del dataset original y/o consultas SQL.
- El archivo en `processed` es el utilizado por Power BI.
