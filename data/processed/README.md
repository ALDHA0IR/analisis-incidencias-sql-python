# Processed Data

Datos procesados y listos para análisis.

## Contenido
- Incidencias filtradas (solo atendidas)
- Columnas limpias y tipadas
- Preparado para consumo en Power BI

## Origen
Este dataset se genera mediante scripts/notebooks en Python a partir de:
- Base de datos MySQL
- Dataset original (raw)

## Uso
- Visualización en Power BI
- Análisis exploratorio
